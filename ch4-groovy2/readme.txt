Very simple example of testing a groovy implementation of a java interface using a groovy test class
The baseline is the model part of the Beer recommendation webapp in chapter 4 of head First Servlets and JSP

Demonstrates 
-implementing a java interface by means of a groovy class
-unit test of groovy code implemented in groovy
-parameterized unit test of groovy code implemented in groovy

In order to run test:
gradle clean check
